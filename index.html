static关键字学习

发表于 2020-03-20 | 分类于 Java基础
  1. static定义的变量是类变量，即可以通过 类.变量 的方式进行访问，static定义的变量不属于任何对象，属于该类，任何对象都可以调用该变量，但是只有那一份
  2. static定义的函数是类函数，他不属于任何对象，属于这个类，
  3. static的函数只能调用static的函数，只能访问static的变量，static的变量和函数能通过类的方式访问，也可以通过对象的形式访问，不过用对象形式访问时不能获得对象的具体信息，
  4. static的变量和函数在类的装载（进入类过程）的时候被初始化，static的初始化和对象的创建无关。

Interface

发表于 2020-03-20 | 分类于 Java基础

接口（interface）

接口是纯抽象类
所有的成员函数都是抽象函数
所有成员变量都是public static final
接口规定了长什么样，但是不管里面有什么
接口是一种class，与class的地位是一样的，即所有能使用class的地方都可以使用interface

实现接口

  • 类用extends，接口用implements
  • 类可以实现很多接口
  • 接口可以继承接口，但不能继承类
  • 接口不能实现接口

面向接口的编程方式

  • 设计程序时先定义接口，再实现类
  • 任何需要再函数间传入传出的一定是接口而不是具体的类
  • 多人合作编程较为有利
  • 代码量快速膨胀，受到批评

本地变量和成员变量

发表于 2020-03-20 | 分类于 Java基础

本地变量和成员变量

  1. 定义在函数内部的变量为本地变量，定义在函数外面的变量是成员变量
  2. 本地变量的作用域和生存期都在函数内部
  3. 成员变量的作用域是类内部，生存期是对象的生存期

class and object

发表于 2020-03-20 | 分类于 Java基础

类和对象的理解

  1. 类定义了对象长什么样，对象则是按照类的定义所制造出来的实体，一个类可以创建很多对象，每个对象有自己的数据

  2. 对象是实体，需要被创建，可以为我们做事前，类是规范，根据类的定义来创建对象

  3. 对象=属性+服务（数据：属性或状态。操作：函数）

  4. 封装：把数据和对数据的操作放在一起

  5. 类定义了对象中所具有的变量，这些变量称作为成员变量，每个对象都有自己的变量，和同一个类定义的其它对象是分开的

  6. 在函数中可以直接写成员变量的名字来访问成员变量，而对于函数的调用则是通过对象进行的
