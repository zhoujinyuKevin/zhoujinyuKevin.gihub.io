Vim三种模式

发表于 2020-04-13 | 分类于 Vim | 本文总阅读量 次
字数统计: 字 | 阅读时长 ≈ 分钟
简介：之前已经介绍过了，Vim是程序员行业最受欢迎的一种文本编辑器，但是Vim在日常使用过程中与常见的编辑器有部分不同，其中最直观的便是其三种模式——一般模式、编辑模式和命令模式，并且Vim的操作都是在命令行下进行的，下面分别对三种模式进行简要介绍并记录一下常用的命令：

一般模式：

当我们用Vim打开一个文件进行操作的时候，默认进入该文件的一般模式，打开文件的命令有很多，如下所示：

命令 作用
vim n filename 打开文件，将光标置于第n行首
vim filename 打开文件，将光标置于追后一行行首
vim /pattern filename 打开文件将光标置于第一个与pattern匹配之处
vim -r filename 上次使用vim编辑器时发生系统奔溃，恢复filename
vim filename……filename 打开多个文件，依次进行编辑，按“n”切换到下一个，按“N”切换到前面一个
在一般模式下，可以进行的操作有：上下移动光标、删除指定字符、删除某行以及负值粘贴一行或者多行，在一般模式下光标移动的命令如下：
按键 作用
h键 左移
l键 右移
k键 上移
j键 下移
Ctrl+u 屏幕向前移动半页
Ctrl+d 屏幕向后移动半页
Ctrl+f或PageUp 屏幕向前移动一页
Ctrl+b或PageDown 屏幕向后移动一页
0或shift+6 移动到本行行首
shift+4 移动到本行行尾
gg 移动到行首
G 移动到行尾
nG 移动到第n行
在一般模式下，我们还可以实现字符串的复制、删除、粘贴等操作：
按键 作用
x 向后删除一个字符
X 向前删除一个字符
nx 向后删除n个字符
dd 删除、剪切光标所在的那一行
ndd 删除、剪切光标所在行后的第n行
YY 复制光标所在行
P 光标所在行开始，向下粘贴已经复制的内容
p 光标所在行开始，向上粘贴已经复制的内容
nyy 光标所在行开始，向下复制n行
u 还原上一步操作
v 按v后光标可以选中指定的字符，可实现复制粘贴
#### 编辑模式：
一般模式不能进行字符的修改，相应的操作只能在编辑模式下进行，一般模式进入编辑模式只需要按i、I、a、A、o、O、r和R中的某一键即可。当进入编辑模式，屏幕的尾行会出现“插入”字样。从编辑模式退回到一般模式，需要按“ESC”，具体命令如下：
按键 作用
i 在当前字符前插入
I 在光标所在行的行首插入
a 在当前字符后插入
A 在光标所在行的行尾插入
o 在当前行的下一行插入新的一行
O 在当前行的上一行插入新的一行
P 光标所在行开始，向上粘贴已经复制的内容
nyy 光标所在行开始，向下复制n行
u 还原上一步操作
#### 命令模式：
在一般模式下，输入“:”或者“/”即可进入命令模式，在该模式下，我们可以进行搜索、保存、替换、退出、显示行号等操作命令如下：
命令 作用
/word 光标之后查找字符串word，按n向后搜索
?word 光标之前查找字符串word，按n向前搜索
:n1,n2/word1/word2/g 在n1-n2行之间查找word1并替换为word2，不加g只替换每行的第一个word1
:1,$s/word1/word2/g 将文档中所有的word1替换为word2，不加g只替换每行的第一个word1
命令模式的其它（如退出），如下表是：
命令 作用
:w 保存文本
:q 退出vim
:w! 强制保存，在root权限下，文件只读也可保存
:q! 强制退出，所有改动不生效
:wq 保存并退出
:x 保存并退出
ZZ 等同于:wq
:set nu 显示行号
:set nonu 不显示行号
