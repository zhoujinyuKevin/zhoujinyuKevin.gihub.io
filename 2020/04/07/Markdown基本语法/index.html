Markdown基本语法

发表于 2020-04-07 | 分类于 Markdown | 本文总阅读量 次
字数统计: 字 | 阅读时长 ≈ 分钟

Markdown是编程行业内，程序员最常使用的文本编辑方式，正因为在程序员中使用较多，不乏各种优秀的编辑器，Markdown编辑方式最显著地特点是通过语法的编写能够给你呈现出不一样的格式，而不像主流的Word等编辑器一样，需要后期对格式进行调整，虽然支持的格式比较简单，但是正是这种简便使得其在程序员之间大受欢迎，另一个比较主要的优点是markdown能进行实时预览，即可以进行左边编辑右边预览的效果，这给我们一种比较直观的格式呈现，而不是像Word那样后期在进行格式的调整。
